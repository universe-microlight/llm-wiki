# 自旋电子学

## 核心原理

用电子的**自旋态**（↑/↓）而非电荷来表示信息。自旋翻转不涉及电荷长距离移动，不碰撞晶格，不产热。

## 关键区分：自旋 ≠ 量子叠加态

| 属性 | 量子叠加态 | 电子自旋 |
|------|-----------|---------|
| 能否观测 | 测量会坍缩 | 可以（磁阻效应读电阻） |
| 怎么读出 | 特殊测量协议 | GMR/TMR 电阻变化 |
| 商用产品 | 量子计算机（极早期） | MRAM 已量产多年 |

自旋向上 ↑ 和自旋向下 ↓ 是经典确定态，通过磁阻效应直接测量。

## 读写机制

### 读出：磁性隧道结（MTJ）

```
固定层（自旋方向固定）
─────────────────
隧道势垒层（~1nm MgO）
─────────────────
自由层（自旋方向可翻转）
```

- 平行 → 低电阻 → 读出 1
- 反平行 → 高电阻 → 读出 0
- 读出就是测电阻，跟读传统内存一样

### 写入：三种机制

| 机制 | 原理 | 特点 |
|------|------|------|
| STT（自旋转移矩） | 自旋极化电流直接翻转自由层 | 读写共用通道，结构简单 |
| SOT（自旋轨道矩） | 重金属层（Pt/W）的自旋霍尔效应 | 读写分离，速度更快 |
| VCMA（电压控制磁各向异性） | 电场调制磁稳定性 | 超低功耗，仍在研究 |

## 产品现状

### MRAM（已量产）

- Everspin、三星、台积电均量产
- 三星 eMRAM：28nm/14nm 嵌入式
- 台积电 eMRAM：22nm/16nm 嵌入式
- 已嵌入先进制程芯片作为缓存（替代部分SRAM）

### 自旋逻辑门（实验阶段）

- 全自旋逻辑门（All-Spin Logic）：纳米磁铁之间的自旋流耦合
- 自旋波逻辑（Magnon Logic）：磁子（自旋波量子）传播
- 2023年 MIT 做出完整自旋基加法器

## 5nm 制程可行性

### MTJ 尺寸

- 当前量产：~20nm 直径
- 实验室最小：~5nm 直径（论文验证）
- 5nm MTJ 本身可以做

### 三个硬约束

**① 磁稳定性 vs 热噪声**

热稳定性 Δ = KV/kT，5nm MTJ 的 Δ ≈ 40-50，10年数据保持需要 Δ > 60。**勉强够但余量很小。**

**② 写入电流密度**

5nm 截面积极小 → 电流密度极高 → 可能击穿隧道势垒层。SOT 方案可以缓解（读写分离）。

**③ 跟3D堆叠兼容性（最关键）**

| 挑战 | 详情 |
|------|------|
| 键合温度 | MTJ 磁性材料高温退化，需 <300°C 低温键合 |
| 层间热预算 | MgO 势垒层对热敏感，多次高温劣化 |
| 互连密度 | 磁性材料跟铜互连工艺不完全兼容 |
| 面积效率 | MTJ 面积比 MOSFET 大，同样面积逻辑密度低 |

### 时间线判断

| 目标 | 时间 | 可能性 |
|------|------|--------|
| MRAM 嵌入 5nm 制程 | 2027-2028 | 高 |
| MRAM 做 3D 堆叠中间层存储 | 2028-2030 | 中高 |
| 自旋逻辑门替代部分 CMOS | 2030+ | 中 |
| 全自旋 5nm 3D 芯片 | 2035+ | 低（需重大突破） |

## 最现实的路径：混合方案

```
第1层（顶层）：传统 CMOS 5nm 逻辑 ← 算力
第2层（中间）：STT-MRAM 替代 SRAM  ← 存储，不产热
第3层（底层）：传统 CMOS 5nm 逻辑 ← 算力
```

**MRAM 替代 SRAM** 是当前最现实的切入点：
- SRAM 占芯片面积 60-70%，是产热大户
- MRAM 面积更小（1个MTJ+1个晶体管 vs SRAM的6个晶体管）
- MRAM 非易失，不刷新不耗电
- 三星/台积电路线图已有

## 评价

自旋电子学是**最接近商用的低功耗方案**。MRAM 已经量产，不需要等物理突破。但用自旋逻辑**全面替代** CMOS 做 5nm 3D 堆叠芯片，目前不太现实。各取所长的混合方案是 3-5 年内能看到的路径。
